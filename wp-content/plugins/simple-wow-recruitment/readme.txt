= Plugin's Official Site =

Simple Wow Recruitment [(http://screennetz.de/develop/simple-wow-recruitment/)](http://screennetz.de/develop/simple-wow-recruitment/)

* [FAQ](http://screennetz.de/develop/simple-wow-recruitment/)

== Installation ==

1. Das komplette Plugin Verzeichnis "simple-wow-recruitment" in das Wordpress Plugin-Verzeichnis "/wp-content/plugins/" hochladen.
2. Plugin über das Admin-Backend aktivieren (Plugins).
3. Unter Design > Widgets können nun x-beliebige Recruitments erstellt werden.
